# CipherSQLStudio

CipherSQLStudio is a browser-based SQL learning platform where users can practice SQL queries on pre-configured assignments with real-time execution and guided hints.

## Tech Stack
- Frontend: React.js, SCSS, Monaco Editor
- Backend: Node.js, Express.js
- Database: PostgreSQL
- AI: Conceptual hint system (no direct SQL solutions)

## Features
- View SQL assignments
- Write and execute SELECT queries
- Secure query validation (only SELECT allowed)
- Real-time result display
- Conceptual AI hints
- Mobile-first responsive UI
